「Webコンテンツ作成」の成果

本テーマの学習目的

本テーマでは、各回の講義を通して HTML, CSS, JavaScript の基本を学びます。これらを組み合わせて Web ページを構成する方法を学びます。最終目標はオリジナリティのあるプロジェクト課題を完成させることです。

プロジェクト課題について

プロジェクト課題では、WWW のサーバーへ検索し、サーバから届いたデータを Web ページとして整形するような Web アプリケーションを作成します。

データは次の3種類で、そのいずれかのデータを検索、結果表示できるようにします。

  • NHk の番組表
  • グルメ情報
  • 世界の天気

作成した「世界の天気」のページ:世界の天気

以下のことが評価されます

  1. ページが正しく動作するか
  2. HTML, CSS, JavaScript の学習の成果がページに反映されているか
  3. 検索機能は使いやすいか
  4. 検索結果は見やすく表示されるか
  5. 検索にヒットした項目がすべて表示されるか
  6. 技術的な面でのオリジナリティがあるか

    苦心点

    第1回の講義から VS Code の不調により、授業に置いてけぼりになるのではないのかという不安でいっぱいの始まりでした。しかし、SAやSEの方の助けで第2回からは普通に授業に参加できて良かったです。

    次に大変だと感じたのは、「数当て」のプログラムです。JavaScriptでは、Javaとは少し勝手が違い if 文の書き方やコンソールから Webページに表示するのにとても苦労し、何度も講義資料を読み直し完成させました。

    最後に大変だったのは、各回の課題で「プロジェクト課題」を少しずつ進めることです。講義資料を読んでも何をすれば良いのか分からず、完成形が見えなかったからです。最後の講義でやっと理解出来て、オリジナリティも出せたと思うので良かったです。

    努力した点、工夫点

    初めてやることで内容も難しく、授業中に課題を終わらすのが難しかったので、次の週までに終わらせて課題を貯めないよう頑張りました。

    また、 MDN Docs も使ってより高度な技術を 「このページ」 や 「プロジェクト課題」 で応用しました。

    強調点

    Web ページに合った見やすいレイアウトやわかりやすく使えるような Web ページになるよう考え作成しました。
